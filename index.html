Getting Started in 3 Steps

1

Download the Project

Get the project files (including the GUI and backend scripts) from our official GitHub repository.

Go to GitHub
2

Install Dependencies

Open your terminal and run the following command to install the required packages:

pip install psutil tqdm reportlab pywin32
3

Run the GUI

Navigate to the project directory and execute the GUI script using Python:

python secure_wipe.py
